## 🧑‍💻 Getting Started

Clone the repository:

```bash
git clone https://github.com/mithlesh2412/Portfolio.git
cd Portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
